# 📦 MCP Gateway v0.3.4

> Turn your APIs into MCP endpoints — effortlessly, without modifying the original code.

## ✨ 新功能

- 支持form-data格式的请求
- 为Go模板添加了新的函数，支持多种方式提取和转换请求/响应数据
- 添加了微信二维码
- 添加了star history展示

## 🔧 改进

- 重构了Docker镜像配置打包方式
- 支持Docker容器的时区设置
