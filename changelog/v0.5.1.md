# 📦 MCP Gateway v0.5.1

> Turn your MCP Servers and APIs into MCP endpoints — effortlessly, without modifying the original code.

## ✨ 新功能

- 添加代理配置的修订历史限制功能
- 添加聊天会话管理功能，包括删除和更新标题
- 为 Redis 中的键添加 TTL 支持

## 🐛 修复

- 修复 header 参数问题
- 回滚 web 基础镜像
