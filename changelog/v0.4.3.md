# 📦 MCP Gateway v0.4.3

> Turn your MCP Servers and APIs into MCP endpoints — effortlessly, without modifying the original code.

## ✨ 新功能

- 支持OpenAPI 3.1和Swagger 2.0转换功能
- 添加用户友好的网关配置编辑模式
- 添加Redis会话存储的前缀配置
- 添加issue模板，支持bug报告、功能请求、讨论和提案
