# 📦 MCP Gateway v0.4.4

> Turn your MCP Servers and APIs into MCP endpoints — effortlessly, without modifying the original code.

## ✨ 新功能

- 添加MCP服务器启动策略配置
- 添加MCP服务器导出功能
- mock服务支持STDIO和SSE
- 添加zod依赖用于模式验证

## 🔧 优化

- 优化MCP代理相关代码
- 简化传输实现，移除不必要的MCP服务器配置

## 🐛 修复

- 添加MCPConfig的json标签，并统一小写驼峰
